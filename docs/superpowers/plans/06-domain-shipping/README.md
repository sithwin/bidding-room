# 06 — Domain: Shipping

## Scope
Backend: Shipping Service (Hono) — fulfilment records, shipping address capture, collection slot booking, fulfilment status tracking.
Frontend: post-payment fulfilment page — choose ship or collect, address form, slot picker.

## Prerequisites
`00-infrastructure`, `01-shared-packages`, `05-domain-payment` complete.
Consumes `PaymentReceived` event from Payment Service.
