# 07 — Domain: Notification

## Scope
Backend only: Notification Service (Hono) — email via Resend + React Email templates, SMS OTP via Twilio.
No frontend — purely event-driven, no HTTP endpoints exposed to clients.

## Prerequisites
`00-infrastructure`, `01-shared-packages` complete.
Can be wired incrementally — subscribes to events from each domain as they ship.
